DynamicUniform

DynamicVertexData

This structure wraps [PerImageDynamicVertexData], managing multiple instances and providing an easy-to-use interface for having per-image buffers. This is needed because multiple images (frames) can be in flight at any given time, so multiple buffers are needed for the same data.

EnvironmentSub

Submodule for loading and binding descriptor sets for a 3D, lit environment. This also abstracts away the need for handling multiple images in flight, as it provides per-image submissions.

FlatEnvironmentSub

Submodule for loading and binding descriptor sets for a flat, unlit environment. This also abstracts away the need for handling multiple images in flight, as it provides per-image submissions.
