pub struct Orbit { /* fields omitted */ }

Represents a Keplerian orbit.

This struct represents the trajectory of a Keplerian orbit in a star system. This is isomorphic to an OrbitalState given a specific mass and time frame.

Extra data are stored in this struct for efficient computation.

pub fn when_intersect_radius(
    &self,
    radius: Length,
    after: GameInstant
) -> Option<GameInstant>[src]

Computes the time (starting from after) when this orbit intersects with the circle of radius radius (either moving into the circle or out of it).

This method only computes a lower bound, i.e. from after up to the returned value, the orbit must not cross the radius circle. This method is not guaranteed to return the ideal lower bound, but is designed to converge towards the intersection efficiently.

This method returns None if the orbit can never, or not in computationally relevant time, intersect with radius.
